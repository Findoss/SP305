## Помочь проекту можно, но не обязательно

Установка

```
git clone https://github.com/aengorg/mf.messenger.praktikum.yandex.git

cd mf.messenger.praktikum.yandex/

npm run install:all
```

Запуск

```
npm run client:dev
```

```
npm run server:dev
```

```
open http://localhost:4000/public/#
```
